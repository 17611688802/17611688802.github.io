开发调试工具
● 查找 Ctrl + P
○ ：+行号 -定位到具体的行
○ @+函数 -定位到具体的函数
○ #+关键字 -匹配到具体的关键字
● 命令面板Ctrl + Shift + P
● 多个选择Ctrl + D，Ctrl + K
○ Alt+F3选中多个
○ Ctrl + Shift + L选中多行

常用插件
Package Control-管理工具
Emmet-编辑HTML和CSS代码
DocBlockr-产生函数的注释
SideBarEnhancements-右键菜单增强工具
Terminal-命令行终端
